How to use

You can add .k-cursor-{name} class in the element.

Examples

Move
Pointer
Zoom in 
<div class="k-cursor-move">
    Move
</div>
<div class="k-cursor-pointer">
    Pointer
</div>
<div class="k-cursor-zoom-in">
    Zoom in
</div>
