Scrollbar

Examples for using Kukun's scrollbar.

Source code

Vertical
Horizontal
.k-scroll-element-v {
  overflow-y: scroll;
  &::-webkit-scrollbar {
    width: 8px;
  }
  &::-webkit-scrollbar-track {
    background: #f7f7f7;
  }
  &::-webkit-scrollbar-thumb {
      background: #999999;
      border-radius: 1px;
      border: 2px solid #f7f7f7;

      &:hover {
        background: #c4c4c4;
      }
  }
}
.k-scroll-element-h {
  overflow-x: scroll;
  white-space: nowrap;

  &::-webkit-scrollbar {
    height: 8px;
  }

  &::-webkit-scrollbar-track {
    background: #f7f7f7;
  }

  &::-webkit-scrollbar-thumb {
    background: #999999;
    border-radius: 1px;
    border: 2px solid #f7f7f7;

    &:hover {
      background: #c4c4c4;
    }
  }
}

How to use

You can add .k-scroll-element-v or .k-scroll-element-h in your element.
